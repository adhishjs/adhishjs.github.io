Low Power PCG Classification Using 1D CSNN

Jan 2026 – Present

Academic Project Guide: Dr. Suresh Balanethiram
Quick Links

Tools & Stack

Python PyTorch NumPy Signal Processing 1D CSNN PhysioNet/CinC 2016

Overview

This project targets low-power heart sound classification by combining compact neural architectures with robust time-frequency front-end representations. The model is designed for deployability on constrained hardware, while preserving high diagnostic quality across normal and abnormal PCG classes.

  • Designed and optimized a compact 1D CSNN pipeline for binary PCG classification.
  • Evaluated CQT, Mel Spectrogram, Chirplet, CWT, and MFCC feature pipelines with identical training settings.
  • Analyzed overfitting behavior, sensitivity/specificity trade-offs, and confusion trends to choose a deployment-oriented configuration.

Training Pipeline

STEP 01

Signal Conditioning

PCG recordings are cleaned, normalized, and segmented into fixed-length windows.

STEP 02

Feature Generation

Five time-frequency transforms are generated and benchmarked under a common setup.

STEP 03

1D CSNN Training

A low-parameter convolutional spiking-inspired model is trained with early stopping.

STEP 04

Model Selection

Best model is chosen using accuracy, sensitivity, specificity, and confusion behavior.

Dataset & Model Snapshot

Dataset PhysioNet/CinC 2016 Heart Sound Dataset
Task Binary PCG classification (Normal / Abnormal)
Model Compact 1D CSNN
Parameter Budget Approximately 16K parameters
Best Feature Family CQT representation
Best Validation Accuracy 96.8%
