Overview

PRC‑PSS (Precision Remote‑Controlled Perimeter Security System) is a hardware‑accelerated, AI‑assisted defence prototype built on the Microchip PolarFire SoC Icicle Kit. It identifies a hostile face in real‑time, computes its position, and aligns a servo‑mounted laser pointer directly at the detected face — all under human‑in‑the‑loop control.

  • SCRFD face detection — lightweight, high‑accuracy model deployed on the FPGA fabric using the VectorBlox CNN IP.
  • Hardware acceleration — the RISC‑V CPU handles video capture, preprocessing, and control, while the FPGA performs low‑latency inference.
  • Precise servo control — real‑time pan‑tilt adjustment based on detected face coordinates.
  • VLSID 2026 Finalist — selected among top projects in the design contest.

System Architecture

Hardware/software co‑design: RISC‑V CPU handles high‑level tasks, FPGA fabric accelerates SCRFD inference.

  • RISC‑V Processor — frame capture, preprocessing, coordinate extraction, servo angle computation, video streaming, human‑interface control.
  • FPGA Fabric — executes the SCRFD face detection model with VectorBlox IP, minimising latency.
  • Yocto Linux — custom image with libjpeg, OpenCV, libgpiod, V4L2 drivers, and FPGA CNN acceleration support.

Results

  • Real‑time face tracking — bounding boxes, centroids, and laser alignment updated per frame.
  • Remote streaming — soldiers monitor the scene from a safe position via Ethernet.
  • Human‑in‑the‑loop — laser activates only on confirmed detection; mandatory soldier approval ensures ethical control.
  • Precision aiming — servo motors with PWM control achieve accurate pan‑tilt alignment.
