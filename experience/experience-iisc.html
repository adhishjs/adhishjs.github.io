Overview

TCAD TCL Script Generator

Developed a Python-based TCL script generator for Sentaurus Visual that automates the extraction of 1D data curves from complex 3D and 2D TCAD simulation files (.tdr) and PLT transient data.

  • Supports 3D analysis (cutplane → cutline → 1D plot), 2D analysis (direct cutline), and PLT-based transient plotting.
  • Automates repetitive manual steps — loading files, creating plots, linking axes, and exporting data.
  • Handles multi-node, multi-TDR, and multi-cutline batch processing with parameter short-code mapping.
  • Provides both terminal-based and GUI-based versions for flexibility.

Tool Architecture

Modular Command Generator

The tool accepts high-level user parameters (Node IDs, Cut locations, Parameters) and algorithmically generates a complete, error-free TCL script for Sentaurus Visual.

Mode 13D: Cutplane → Cutline → 1D Plot
Mode 22D: Direct Cutline → 1D Plot
Mode 3PLT: Transient analysis with CSV/PNG export
OutputReady-to-run TCL script for Sentaurus Visual

Key Features

Multi-Node & Multi-TDR

Process multiple simulation nodes and TDR files in batch — each with its own parameter values — all from a single configuration.

Multi-Cutline Handling

Define multiple cut sets (straight or L-shaped) across different planes and axes, with independent parameter selections for each.

Parameter Short-Code Mapping

Uses a dictionary to abbreviate long parameter names (e.g., DopingConcentration → DC), keeping generated TCL scripts clean and readable.

GUI & Terminal Versions

Offers both a user-friendly GUI (built with Tkinter) for interactive use and a terminal-based version for scripting and automation.
