Overview

Spacecraft Design Suite

Developed a comprehensive Python-based spacecraft design environment for rapid system-level trade-off evaluation across multiple spacecraft subsystems.

  • Integrated power budgeting, communication link analysis, hardware power estimation, data budgeting, and memory sizing into a unified desktop application.
  • Built modular calculation engines for subsystem analysis so engineers can compare design options quickly.
  • Provided automatic recalculation, editable input tables, and export-oriented workflows for design reviews and reporting.

Tool Architecture

Modular Budget Engine

The tool is organized as a modular desktop application with separate calculation domains for power, link, hardware, data, and memory budgets.

Power BudgetSolar array and battery sizing
Link BudgetUHF and transponder analysis
Hardware PowerBoard and circuit load estimation
Data BudgetTelemetry and daily volume
Memory BudgetStorage and rollback sizing
OutputAuto-updating design summary

Features

Power Budget

Supports orbit parameters, solar array sizing, battery sizing, component loads, energy balance, depth of discharge, and recharge time checks.

UHF Link Budget

Models uplink/downlink frequencies, transmit power, antenna gains, path losses, atmospheric attenuation, CNR, link margin, Doppler shift, and slant range.

Transponder Link

Calculates uplink and downlink C/N, combined S/N, achievable data rate, and spectral efficiency using adaptive modulation lookup.

Hardware, Data, & Memory

Summarizes board power, telemetry rates, daily data volume, and storage sizing with power-of-two recommendations for NVM, RAM, and rollback allocation.
